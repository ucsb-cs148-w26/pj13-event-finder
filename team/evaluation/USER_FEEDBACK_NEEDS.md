# Event Finder User Feedback

## User Feedback Goals
We want user feedback on:

- How satisfied the user is when using the overall app
- If the filter system spans over a satisfactory amount of categories
- How easily the user could use the app
- How good the current UI design is
- If the current system for displaying search results is satisfactory to the user
- Any potential bugs

---

## User Feedback Metrics

- How many of the given filters would you use or find useful?
- How many events displayed were interesting to you?
- How long did it take you to figure out the UI works?

---

## Metrics to Measure Improvement

### Objective

Increase functionality and user satisfaction with the app

### Measureable Metrics

1. Achieve an average metric of:
   - User finds at least 1 filter option useful
   - User finds at least 1 displayed event interesting
   - User was able to proply figure out how to use the UI in under 1 minute

2. Testers can use the app without experiencing bugs.
